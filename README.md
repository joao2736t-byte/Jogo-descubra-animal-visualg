#Jogo-descubra-animal-visualg

Um jogo de adivinhar um animal no VisualG com 28 animais diferentes. Você precisa responder às perguntas de sim ou não até que um determinado animal (o que você escolheu) seja descoberto.

##Documentação Técnica

É bastante simples, escrito no VisualG (simplesmente informe qual animal o usuário está pensando em algumas perguntas de “Sim” ou “Não”).

**Lógica usada:**

Ele tem um menu principal e então começa a fazer perguntas ao usuário, por exemplo. Apenas respostas “Y” ou “N”.

**Estruturas aplicadas:**

Métodos para separar seções de código (menu, instruções, cabeçalho, arte ASCII).

Laços de repetição (repita...até) para validar entradas.

Estruturas condicionais (se...então...senão). Acertar ou errar sua resposta determinaria o fluxo das perguntas.

**Limitações conhecidas:**

Respostas simples SIM (“S”) e NÃO (“N”).

A lista de animais é fixa (28 espécies)

##Manual de utilização

Instale o VisualG no seu computador.

Baixe o arquivo Adivinhador-de-animais.ALG deste repositório.

Você ira abrir o VisualG, clicar no arquivo → abrir e escolher o código que você baixou.

Clique em Executar → Executar Algoritmo ou pressione F9 para executá-lo.

Nas solicitações na tela, siga as instruções e responda S (Sim) ou N (Não).
